# 单例模式
一个类只有一个实例，且该类能够自行创建这个实例的模式
# 特点
1. 单例类只有一个实例对象，节省内存空间。
2. 该单例对象必须由单例类自行创建，同时避免频繁的创建销毁对象，可以提高性能。
3. 单例类对外提供一个访问该单例的全局访问点。
# 单例适用的场景
1、需要频繁实例化然后销毁的对象。
2、创建对象时耗时过多或者耗资源过多，但又经常用到的对象。
3、有状态的工具类对象。
4、频繁访问数据库或文件的对象。
